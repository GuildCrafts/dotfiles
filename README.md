# Dot Files


This repository contains example bash configuration files to help get you
started.

## Usage

Copy these files to these locations:

| File in this repo | Destination on your computer |
| ----------------- | ---------------------------- |
| bash_profile      | ~/.bash_profile              |
| gitignore         | ~/.gitignore                 |
| gitconfig         | ~/.gitconfig                 |
| bashrc            | ~/.bashrc                    |
